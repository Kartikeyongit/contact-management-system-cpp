# Contact Management System (C++)

A simple Contact Management System built using C++.  
Users can add, view, search, update, and delete contacts.  
All contacts are stored in a text file using C++ file handling.

## Features
- Add new contact
- View all contacts
- Search contact by name or number
- Update existing contact
- Delete contact
- Persistent storage using contacts.txt

## How to Run
g++ main.cpp -o contact
./contact

## Folder Structure
contact-management-cpp/
│── main.cpp
│── contacts.txt (auto-created)
